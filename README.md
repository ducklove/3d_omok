# Prism Omok 3D

화려한 3D 연출, AI 대전, 힌트, 리플레이, 타이머, 자동 저장을 포함한 정적 웹 오목 게임입니다.

## 실행

```bash
cd /home/cantabile/Works/3d_omok
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 를 열면 됩니다.

## 포함 기능

- Three.js 기반 3D 보드와 수동 회전/줌 카메라
- 깔끔한 실좌표 격자 오버레이, 완만한 원근 카메라, 테마 전환
- 목재 타격감의 착수음, 힌트/승리/시간 종료 사운드 효과
- 파티클, 승리 라인, 마지막 수 강조
- 로컬 2인 / AI 대전, Easy/Normal/Hard 난이도
- 힌트, 되돌리기/다시 실행, 리플레이, 진영 전환
- 11x11 / 15x15 / 19x19 보드
- 자유 규칙 / 정확히 5 규칙
- 턴 타이머, 전적 저장, 경기 상태 자동 복원
